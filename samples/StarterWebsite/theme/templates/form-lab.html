What this demo covers

  • Server-rendered Bootstrap form HTML from the C# form definition
  • Client validation generated from the C# form model
  • Server binding through request.Forms.BindAsync<T>()
  • Structured field errors
  • Sections, grid spans, hidden/read-only fields, provider-backed select fields, and async autocomplete search
  • File size, content type, extension, and image dimension checks

Schema and render source

The page embeds a client schema generated from the same C# model the server renders, binds, and validates. The favorite-color options are overridden by the server at render time.
